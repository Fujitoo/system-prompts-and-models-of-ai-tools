# AI Agent Finder 🤖

A modern web application built with Vite and React that helps developers find the best AI coding assistant for their specific tasks.

## Features

✨ **Task-Based Recommendations** - Select your use case and get personalized AI agent recommendations

🔍 **Smart Search** - Search agents by name, features, or use cases

⚖️ **Side-by-Side Comparison** - Compare up to 3 AI agents simultaneously

📊 **Comprehensive Database** - Detailed information about 10+ leading AI coding assistants including:
- Claude Code
- Cursor
- Devin AI
- GitHub Copilot
- Windsurf
- Augment Code
- Replit Ghostwriter
- v0 by Vercel
- Perplexity Pro
- Trae

❓ **FAQ Section** - Answer Engine Optimized content to help users make informed decisions

## AEO (Answer Engine Optimization)

This application is optimized for answer engines and AI search results with:

- **Structured Data (Schema.org)** - WebApplication and FAQPage markup
- **Semantic HTML** - Proper heading hierarchy and semantic elements
- **Rich Meta Tags** - Comprehensive meta descriptions, Open Graph, and Twitter cards
- **Natural Language Content** - FAQ section written in question-answer format
- **Keyword Optimization** - Strategic use of relevant keywords throughout

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
# Navigate to the project directory
cd ai-agent-finder

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

The production build will be available in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```
